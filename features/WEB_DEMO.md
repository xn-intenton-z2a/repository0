# WEB_DEMO

## Summary
A small interactive web demo under src/web/ that uses the library to render results for interactive verification and visual testing.

## Behavior
- Provide a page with an integer input for n and a button to render results.
- Show results in a scrollable list and provide a copy-to-clipboard action.
- The demo must import and use the library's named exports rather than reimplementing logic.

## Acceptance criteria
- src/web/ contains a demo page that calls fizzBuzz(100) and displays the results when the button is clicked.
- The demo uses the library import and shows Fizz, Buzz, and FizzBuzz correctly.
