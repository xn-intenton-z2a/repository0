# Web Demo Interface

Interactive web interface that demonstrates Hamming distance functions with real-time calculation and visualization.

## Requirements

Create engaging web demo in src/web/ that showcases library functionality:

- Interactive input fields for string and integer comparison
- Real-time calculation display as user types
- Visual representation of differences highlighted
- Binary visualization for integer comparisons
- Error display for invalid inputs
- Examples and use cases

## Acceptance Criteria

- Two input sections: string comparison and bits comparison
- String inputs highlight differing characters with color coding
- Integer inputs show binary representation with differing bits highlighted
- Distance calculation updates in real-time
- Clear error messages for validation failures
- Example buttons that populate inputs with demonstration cases
- Responsive design that works on mobile devices
- Behavior tests verify interactive functionality

## User Interface Elements

- Text input pairs for string comparison
- Number input pairs for integer comparison  
- Distance result display with large, clear numbers
- Character-by-character or bit-by-bit difference visualization
- Reset and example buttons for quick testing
- Instructions and documentation links

## Implementation Notes

- Use vanilla JavaScript for simplicity and performance
- Import library functions from relative path to main.js
- Style with modern CSS for clean, professional appearance
- Ensure accessibility with proper labels and keyboard navigation
- Test across different browsers and devices
