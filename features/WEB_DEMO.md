# WEB_DEMO

Summary

Add a small interactive demonstration in src/web/ that uses the exported library functions to render fizzBuzz(15) and allow entering a custom n to update the view.

Specification

- Provide a demo page or component that imports the library and displays the fizzBuzz(15) result on load.
- Include a simple input and button to compute and render fizzBuzz for user-provided values.
- The demo should be included in the built docs so it can be exercised with npm start.

Acceptance Criteria

- [ ] The demo displays the fizzBuzz(15) output on page load and the last element is FizzBuzz.
- [ ] The demo allows entering a value and updates the displayed output accordingly.
- [ ] The project start script builds the web demo and the page is reachable locally via npm start.
