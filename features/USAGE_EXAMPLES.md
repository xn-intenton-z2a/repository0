# USAGE_EXAMPLES

Goal

Document usage examples for library consumers and for README inclusion so the intended behaviours are easy to verify and reproduce.

Description

Provide concise usage examples demonstrating:

- Importing named exports from src/lib/main.js.
- Calling fizzBuzz and fizzBuzzSingle with normal inputs.
- Example showing expected return values for common inputs (3, 5, 15, 7, 0).
- Examples that show throwing behaviour for invalid inputs (negative, non-integer, non-number) — shown as comments explaining expected exceptions.

Acceptance criteria

- README or docs can copy these examples directly without modification.
- Examples are precise (no ellipses) and show exact expected outputs where applicable.
- At least one unit test mirrors an example from this file to ensure documentation stays accurate.

Notes

Keep examples brief and focused; no large code blocks beyond single-call examples. Align phrasing with CONTRIBUTING.md style guidelines.
