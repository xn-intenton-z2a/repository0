# FIZZBUZZ_WEB_EXAMPLE

## Overview
Provide a simple web example page under src/web that uses the library to render FizzBuzz output. This makes the library demonstrable on the repository website and helps manual QA.

## Specification
- Add or update a minimal client-side example in src/web that imports the library and renders fizzBuzz results for a configurable n.
- The page should allow changing n via a query parameter or a simple input and display results in a list.

## Tests
- Add an example HTML file under src/web and include a short manual test plan in README explaining how to build and view the example.

## Acceptance Criteria
- Building the docs (npm run build:web) copies a working example into docs/ that shows FizzBuzz output.
- The README documents how to open and use the web example.
