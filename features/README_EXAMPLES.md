# README_EXAMPLES

## Summary
Examples for the README demonstrating library import and CLI usage without using fenced code blocks.

## Examples
- Library usage:
  Import the functions with import { fizzBuzz, fizzBuzzSingle } from src/lib/main.js and call fizzBuzz(15) to obtain an array of results. Calling fizzBuzzSingle(3) returns Fizz.
- CLI usage:
  Run node src/lib/main.js 15 to print results line by line. Run node src/lib/main.js 15 --json to print a JSON array.

## Acceptance criteria
- README contains at least one library usage example showing how to import and call fizzBuzz and fizzBuzzSingle.
- README contains at least one CLI usage example showing positional and --json usage.
