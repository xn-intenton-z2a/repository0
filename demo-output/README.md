## Command Examples

Each file was generated using commands like:

```bash
# Basic function plot
plot-code-lib plot -e "sin(x)" -r "x=0:2*pi" -o sine-wave.svg --title "Sine Wave"

# Multi-expression comparison
plot-code-lib plot -e "sin(x),cos(x),tan(x)" -r "x=-pi:pi" -o trig-comparison.svg

# Parametric curves
plot-code-lib parametric -x "16*sin(t)^3" -y "13*cos(t)-5*cos(2*t)-2*cos(3*t)-cos(4*t)" -r "t=0:2*pi" -o heart-curve.svg

# Data export
plot-code-lib export -e "sin(x),cos(x)" -r "x=0:2*pi:0.1" -o trig-data.csv --format csv
```
