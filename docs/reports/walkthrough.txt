Walkthrough: Using the Hamming distance library

1. Import the functions from the library entrypoint:
   import { hammingDistance, hammingDistanceBits } from './src/lib/main.js'

2. Compute string Hamming distance (Unicode-aware):
   hammingDistance('karolin', 'kathrin') // => 3

3. Compute bit Hamming distance:
   hammingDistanceBits(1, 4) // => 2

Notes:
- Strings are compared by Unicode code points (Array.from is used internally).
- For precomposed vs combining sequences, lengths may differ and a RangeError will be thrown; normalize inputs if desired with String.prototype.normalize('NFC').
- hammingDistanceBits accepts Number or BigInt; large bit indices are safe when using BigInt.
