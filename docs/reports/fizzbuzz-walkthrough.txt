Walkthrough: Using the FizzBuzz core

Examples:
- fizzBuzzSingle(3) => "fizz"
- fizzBuzzSingle(5) => "buzz"
- fizzBuzzSingle(15) => "fizzbuzz"
- fizzBuzzSingle(7) => "7"
- fizzBuzz(1,5) => ["1","2","fizz","4","buzz"]

Validation policy is strict: functions throw TypeError for non-integer inputs and NaN/Infinity. Ranges where start > end return an empty array.
