# Walkthrough: Hamming Distance Utilities

This report demonstrates the usage of the hammingDistance and hammingDistanceBits functions.

Examples

- hammingDistance('karolin', 'kathrin') → 3
- hammingDistance('', '') → 0
- hammingDistance('a\u{1F600}', 'a\u{1F601}') → 1

- hammingDistanceBits(1, 4) → 2
- hammingDistanceBits(0, 0) → 0

Output examples are available in `docs/examples/example-output.txt` and machine-readable results in `docs/evidence/output.json`.
