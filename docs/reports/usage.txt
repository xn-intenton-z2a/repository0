Cron engine usage examples

parseCron('*/15 0-23/1 * * *') -> parsed object with minutes [0,15,30,45]
nextRun('0 0 * * *', new Date(2026,2,6,0,0,0)) -> 2026-03-07T00:00 local
matches('0 0 25 12 *', new Date(2025,11,25,0,0,0)) -> true

See src/lib/main.js exports: parseCron, nextRun, nextRuns, matches, toString
