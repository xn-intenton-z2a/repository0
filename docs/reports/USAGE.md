Roman Numerals Library — Usage Report

This report demonstrates usage examples and sample outputs from the library.

Examples:

- toRoman(1994) -> MCMXCIV
- fromRoman("MCMXCIV") -> 1994

Round-trip property: fromRoman(toRoman(n)) === n for canonical inputs 1..3999.

Evidence and examples are available under docs/examples/ and docs/evidence/.
