# FizzBuzz Usage

This report demonstrates usage of the fizzBuzz and fizzBuzzSingle utilities.

Examples

- fizzBuzzSingle(3) -> "Fizz"
- fizzBuzzSingle(5) -> "Buzz"
- fizzBuzzSingle(15) -> "FizzBuzz"
- fizzBuzzSingle(2) -> "2"
- fizzBuzz(5) -> ["1","2","Fizz","4","Buzz"]
- fizzBuzz(15) -> see docs/evidence/fizzbuzz.json

Files created by this change:
- src/lib/fizzbuzz.js (exports fizzBuzzSingle and fizzBuzz)
- tests/unit/fizzbuzz.test.js (unit tests)
- docs/examples/fizzbuzz-output.txt (human readable example)
- docs/evidence/fizzbuzz.json (machine-readable output)
