Roman Numerals Library - Usage Report

This report shows sample conversions and demonstrates the round-trip property for selected values.

Samples:
- 1994 -> MCMXCIV -> 1994
- 4 -> IV -> 4
- 3999 -> MMMCMXCIX -> 3999

The library exports: toRoman(n), fromRoman(s), parseRoman(s) and metadata (name, version, description).
