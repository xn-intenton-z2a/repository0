=== String Hamming Distance Examples ===

Mission examples:
hammingDistance("karolin", "kathrin") = 3
hammingDistance("", "") = 0
hammingDistance("a", "bb") throws: Strings must have equal length

Real-world examples:
hammingDistance("hello", "world") = 4
hammingDistance("testing", "resting") = 1

Unicode examples:
hammingDistance("café", "cave") = 2
hammingDistance("🎉🎊", "🎉🌟") = 1
hammingDistance("résumé", "resume") = 2

=== Bits Hamming Distance Examples ===

Mission examples:
hammingDistanceBits(1, 4) = 2 (binary: 001 vs 100)
hammingDistanceBits(0, 0) = 0

Binary analysis:
5 (101) vs 3 (11) = 2 bits
15 (1111) vs 7 (111) = 1 bit
255 (11111111) vs 0 (0) = 8 bits
1024 (10000000000) vs 1 (1) = 2 bits

=== Error Handling Examples ===

String errors:
Different lengths: Strings must have equal length
Wrong type: First argument must be a string

Bits errors:
Negative number: First argument must be non-negative
Non-integer: First argument must be an integer
