# FizzBuzz Demo Documentation

This directory contains the built web demo for the FizzBuzz library.

Usage:
- Open docs/index.html in a browser to interact with the demo.
- The demo uses the library from src/lib/main.js and shows single value and range outputs.

Error messages are exact and used by tests:
- TypeError message: "n must be an integer"
- RangeError message: "n must be a non-negative integer"
