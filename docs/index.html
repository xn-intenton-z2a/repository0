Generate beautiful SVG and PNG plots from mathematical expressions with a simple CLI or web interface. Transform expressions like y=sin(x) into publication-ready visualizations.

✨ Core Features

🧮 Expression Parser

Parse mathematical expressions using standard syntax. Support for common functions like sin, cos, log, and arithmetic operations.

Enter an expression to test parsing...

📈 Time Series Generator

Generate coordinate data from expressions and ranges. Configurable step sizes for smooth curves.

Configure range to generate coordinate data...

⚡ CLI Interface

Full command-line interface with flags for expression, range, and output file specification.

$ node src/lib/main.js --expression "y=sin(x)" --range "x=-pi:pi" --file sine.svg $ node src/lib/main.js --expression "y=x^2+1" --range "x=-3:3" --file parabola.png $ node src/lib/main.js --help

📖 Example Usage

# Generate a sine wave $ node src/lib/main.js --expression "y=sin(x)" --range "x=-pi:pi" --file sine.svg # Create a parabola with custom step size $ node src/lib/main.js --expression "y=x^2-2*x+1" --range "x=-2:4,step=0.05" --file parabola.png # Plot logarithmic function $ node src/lib/main.js --expression "y=log(x)" --range "x=0.1:10" --file log.svg --verbose
