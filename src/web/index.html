Features

  • Expression Parser: Parse mathematical expressions using JavaScript Math functions
  • Range Evaluation: Evaluate expressions over numeric ranges with configurable step sizes
  • CSV Data Loading: Load time series data from CSV files with time,value columns
  • SVG Rendering: Generate scalable vector graphics with polyline elements and viewBox attributes
  • PNG Output: Create bitmap images using HTML5 Canvas (Node.js only)
  • File Saving: Save plots to files with automatic format detection (.svg/.png)
  • CLI Interface: Full command-line interface with --expression, --range, --csv, and --file flags
