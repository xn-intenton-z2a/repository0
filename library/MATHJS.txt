1. Features
- Flexible expression parser with support for symbolic computation.
- Large set of built-in functions and constants.
- Integrated data types: numbers, big numbers, bigint, complex numbers, fractions, units, strings, arrays, and matrices.
- Compatible with JavaScript Math.
- Extensible and works on any JS engine; can be used as a CLI.

2. Expression parser and evaluation
- Core capability: evaluate an expression string into a computed result.
- Example from source: math.evaluate('1.2 * (2 + 4.5)') // returns 7.8
- Unit conversion expression example: math.evaluate('12.7 cm to inch') // returns '5 inch'
- Trigonometric evaluation example: math.evaluate('sin(45 deg) ^ 2') // returns 0.5
- Complex number evaluation example: math.evaluate('9 / 3 + 2i') // returns 3 + 2i

3. Data types supported (explicit list from source)
- Number
- BigNumber
- BigInt
- Complex numbers
- Fractions
- Units (with parsing and conversion support)
- Strings
- Arrays
- Matrices

4. Chaining API (as shown)
- math.chain(value).add(4).multiply(2).done() // chaining operations, returns final value

5. Example invocations (literal extracts from source)
- math.round(math.e, 3) // 2.718
- math.atan2(3, -3) / math.pi // 0.75
- math.log(10000, 10) // 4
- math.sqrt(-4) // 2i
- math.derivative('x^2 + x', 'x') // 2*x+1
- math.pow([[-1, 2], [3, 1]], 2) // [[7,0],[0,7]]
- math.evaluate('det([-1, 2; 3, 1])') // -7

6. Supplementary details (implementation/operational notes from source)
- Math.js exposes evaluate/parse/derivative utilities; expression parser supports units and symbolic derivative.
- CLI available and interactive tools (e.g., Math Notepad) are provided by the project.
- Shortcut behaviors: autocomplete via Tab in the demo.

7. Reference details (literal API usage examples from site — use as authoritative call patterns)
- evaluate(expression: string) => any; examples: '1.2 * (2 + 4.5)', '12.7 cm to inch', 'sin(45 deg) ^ 2'
- derivative(expression: string, variable: string) => parsed derivative expression; example: math.derivative('x^2 + x', 'x') // '2*x+1'
- chain(value: number|BigNumber|Matrix) => Chain object with arithmetic methods and .done() => final value
