# Example Plots

This directory contains example SVG plots generated by plot-code-lib to demonstrate various mathematical functions.

## Generated Examples

### 1. Sine Wave (`sine.svg`)
```bash
plot-code --expression "y=sin(x)" --range "x=-pi:pi" --output examples/sine.svg
```
Classic sine wave from -π to π.

### 2. Quadratic Function (`quadratic.svg`) 
```bash
plot-code --expression "y=x^2" --range "x=-3:3" --output examples/quadratic.svg
```
Simple parabola showing quadratic growth.

### 3. Gaussian/Normal Distribution (`gaussian.svg`)
```bash
plot-code --expression "y=exp(-0.5*x*x)" --range "x=-3:3" --output examples/gaussian.svg
```
Bell curve approximating a normal distribution.

## Tips for Complex Expressions

For expressions involving negative exponents or complex operations, use parentheses to ensure proper evaluation:

- ✅ Good: `y=exp(-0.5*x*x)`
- ❌ Problematic: `y=exp(-x^2)` (can cause parsing issues)

- ✅ Good: `y=sin(2*pi*x)`
- ✅ Good: `y=(x+1)^2 - 4`
- ✅ Good: `y=1/(1+exp(-x))`

## Viewing the SVG Files

SVG files can be opened in:
- Web browsers (Chrome, Firefox, Safari, etc.)
- Vector graphics software (Inkscape, Adobe Illustrator)
- Text editors (to view the SVG source code)
- Embedded in HTML documents or markdown files
