# plot-code-lib Examples Showcase

This directory contains example plots generated by plot-code-lib, demonstrating the range of mathematical expressions and visualizations possible with the tool.

## Basic Mathematical Functions

### Trigonometric Functions
- **`basic_sine.svg`** - `sin(x)` from -2π to 2π
- **`cosine.svg`** - `cos(x)` wave
- **`tangent.png`** - `tan(x/2)` function

### Polynomial Functions  
- **`quadratic_function.svg`** - `x² - 4x + 3` (parabola with roots)
- **`cubic_polynomial.png`** - `x³ - 6x² + 9x - 4` (cubic with inflection points)

### Exponential and Logarithmic
- **`logarithm.svg`** - `ln(x)` natural logarithm
- **`sigmoid.svg`** - `1/(1+e^(-x))` sigmoid function (ML activation)

## Advanced Mathematical Expressions

### Damped and Modulated Functions
- **`damped_oscillation_detailed.png`** - `e^(-x²/8)·cos(2x)` high-resolution damped oscillation
- **`modulated_sine.svg`** - `sin(5x)·e^(-x²/10)` amplitude modulated sine wave
- **`test_damped_cosine.png`** - `cos(x)·e^(-x/5)` exponentially damped cosine

### Composite Functions
- **`absolute_value_composite.svg`** - `|sin(x)| + 0.5·cos(x/3)` combining absolute value and harmonics
- **`complex.svg`** - `sin(x) * cos(x/2)` product of trigonometric functions

### High-Resolution Examples
- **`detailed.svg`** - Complex expressions with 200+ calculation steps for smooth curves

## Command Examples

All examples were generated using commands like:

```bash
# Basic trigonometric
node src/lib/main.js plot --expression "sin(x)" --range "x=-6.28:6.28" --file examples/basic_sine.svg --title "Basic Sine Function"

# Polynomial with custom range
node src/lib/main.js plot --expression "x^2 - 4*x + 3" --range "x=-1:5" --file examples/quadratic_function.svg --title "Quadratic: f(x) = x² - 4x + 3"

# High-resolution complex expression (PNG)
node src/lib/main.js plot --expression "exp(-x^2/8)*cos(2*x)" --range "x=-8:8" --file examples/damped_oscillation_detailed.png --title "Damped Oscillation: e^(-x²/8)·cos(2x)" --steps 300

# Composite functions
node src/lib/main.js plot --expression "abs(sin(x)) + 0.5*cos(x/3)" --range "x=-15:15" --file examples/absolute_value_composite.svg --title "Composite: |sin(x)| + 0.5·cos(x/3)"
```

## Features Demonstrated

- **Multiple Output Formats**: Both SVG (vector) and PNG (raster) examples
- **Custom Ranges**: Different x-axis ranges optimized for each function
- **Variable Resolution**: From standard 100 steps to high-resolution 300 steps
- **Complex Expressions**: Nested functions, exponentials, products, absolute values
- **Mathematical Notation**: Proper handling of special characters in titles
- **Automatic Scaling**: Y-axis automatically scaled to data range

- **Education**: Visualizing mathematical concepts for students
- **Research**: Quick plotting of experimental functions and models
- **Documentation**: Generating plots for papers, presentations, reports  
- **Web Development**: SVG plots for interactive web applications
