# FizzBuzz Library and CLI

This repository implements a small FizzBuzz library and a minimal command-line interface (CLI) to run it.

Examples

- Single value:

  node src/lib/main.js --single 3

  Output:
  Fizz

- Range (1..N):

  node src/lib/main.js --range 15

  Output: prints 15 lines with Fizz/Buzz substitutions; the last line is "FizzBuzz".

See tests in tests/unit for examples of usage.
