# repository0 - FizzBuzz Feature

This repository now includes a small FizzBuzz library and demo.

Usage:

- Run unit tests: npm test
- Run behaviour tests: npm run test:behaviour
- View demo locally: npm start (then open the served site)

Examples:

- fizzBuzzSingle(15) -> "fizzbuzz"
- fizzBuzz(1,15) -> ["1","2","fizz","4","buzz","fizz","7","8","fizz","buzz","11","fizz","13","14","fizzbuzz"]

First 15 outputs for 1..15:

1,2,fizz,4,buzz,fizz,7,8,fizz,buzz,11,fizz,13,14,fizzbuzz
