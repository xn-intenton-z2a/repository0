# Sources

Reference material and documentation sources for this project.

- https://en.wikipedia.org/wiki/Hamming_distance
- https://en.wikipedia.org/wiki/Hamming_weight
- https://en.wikipedia.org/wiki/Population_count
- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/codePointAt
- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/@@iterator
- https://262.ecma-international.org/6.0/#sec-string.prototype-@@iterator
- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/fromCodePoint

- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Numbers_and_dates#bitwise-operators
- https://en.wikipedia.org/wiki/Brian_Kernighan
- https://nodejs.org/api/buffer.html

These sources cover the mathematical definition of Hamming distance, bit-level population count techniques (Hamming weight), efficient popcount algorithms, correct handling of Unicode code points in JavaScript strings (String iteration, codePointAt, fromCodePoint), and bitwise operations useful for integer Hamming distance calculations. The Brian Kernighan link documents an efficient popcount algorithm useful for counting differing bits.
