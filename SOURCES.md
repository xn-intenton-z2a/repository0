# Sources

Reference material and documentation sources for this project.

Add URLs, papers, API docs, or other reference material here.
The maintain-library workflow will process these into `library/` documents.

- https://en.wikipedia.org/wiki/Hamming_distance
  - Authoritative definition, formula, and properties of Hamming distance (string and vector contexts).

- https://en.wikipedia.org/wiki/Hamming_weight
  - Hamming weight (population count) reference for counting set bits when computing bitwise Hamming distance.

- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Bitwise_Operators
  - MDN reference for JavaScript bitwise operators (AND, OR, XOR, shifts) used to compare integer bit patterns.

- https://graphics.stanford.edu/~seander/bithacks.html#CountBitsSetParallel
  - "Bit Twiddling Hacks" — practical algorithms for fast population-count (popcount) and bit manipulation techniques.

- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/codePointAt
  - MDN reference for String.prototype.codePointAt (correct handling of Unicode code points vs UTF-16 code units).

- https://mathiasbynens.be/notes/javascript-unicode
  - Practical guidance on Unicode handling in JavaScript, iterating code points, and pitfalls with surrogate pairs.

- https://www.npmjs.com/package/hamming-distance
  - Example existing npm package and README to compare API shapes and edge-case handling.
