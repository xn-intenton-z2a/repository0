# Sources

Reference material and documentation sources for this project.

- https://en.wikipedia.org/wiki/Hamming_distance
- https://en.wikipedia.org/wiki/Hamming_weight (Popcount / Hamming weight)
- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/codePointAt
- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/@@iterator (String iteration and Unicode code points)
- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Bitwise_Operators

These sources cover the mathematical definition of Hamming distance, bit-level population count techniques, correct handling of Unicode code points in JavaScript strings, and bitwise operations useful for integer Hamming distance calculations.
