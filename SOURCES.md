# Sources

Reference material and documentation sources for this project.

Add URLs, papers, API docs, or other reference material here.
The maintain-library workflow will process these into `library/` documents.

- https://mathjs.org/ — mathjs: extensive math expression parser and evaluator for JavaScript; supports complex numbers, units, expression parsing, and evaluation.
- https://www.npmjs.com/package/expr-eval — expr-eval: lightweight JavaScript expression parser/evaluator suitable for parsing y=sin(x) and similar formulas.
- https://d3js.org/ — D3.js: data-driven documents library providing SVG plotting primitives for building custom visualizations.
- https://vega.github.io/vega-lite/ — Vega-Lite: high-level declarative grammar for interactive visualizations; compiles to Vega for rendering to SVG/PNG.
- https://sharp.pixelplumbing.com/ — Sharp: high-performance Node.js image processing library; renders/converts SVG to PNG and handles resizing and output.
- https://github.com/Automattic/node-canvas — node-canvas: Canvas API implementation for Node.js enabling raster rendering and PNG export.
- https://datatracker.ietf.org/doc/html/rfc4180 — RFC 4180: Common Format and MIME Type for CSV Files; authoritative CSV rules for time-series export/import.
- https://github.com/tj/commander.js — Commander.js: widely used Node.js CLI option parser supporting flags like --expression, --range, --file.
