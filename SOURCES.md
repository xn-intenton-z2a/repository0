# Sources

Reference material and documentation sources for this project.

- https://en.wikipedia.org/wiki/Hamming_distance
- https://en.wikipedia.org/wiki/Hamming_weight
- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/codePointAt
- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/@@iterator
- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Bitwise_Operators
- https://en.wikipedia.org/wiki/Brian_Kernighan
- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/String/fromCodePoint
- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Numbers_and_dates#bitwise-operators
- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Grammar_and_types#unicode

These sources cover the mathematical definition of Hamming distance, bit-level population count techniques (Hamming weight), correct handling of Unicode code points in JavaScript strings (String iteration, codePointAt, fromCodePoint), and bitwise operations useful for integer Hamming distance calculations. The Brian Kernighan link documents an efficient popcount algorithm useful for counting differing bits.
