## Binary-to-Text Encoding References

- https://en.wikipedia.org/wiki/Binary-to-text_encoding - Comprehensive overview of binary-to-text encoding schemes
- https://tools.ietf.org/rfc/rfc4648.txt - RFC 4648: The Base16, Base32, and Base64 Data Encodings (official standard)
- https://en.wikipedia.org/wiki/Base62 - Base62 encoding specification and character set
- https://en.wikipedia.org/wiki/Ascii85 - Ascii85/Base85 encoding specification with efficiency details
- https://rfc-editor.org/rfc/rfc1924.html - RFC 1924: Compact representation using Base85 for IPv6 addresses
- http://base91.sourceforge.net/ - Base91 encoding project with efficiency specifications (14-23% overhead)
- https://github.com/ai/nanoid - NanoID: URL-safe string ID generator using Base64URL alphabet
- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/ArrayBuffer - MDN ArrayBuffer documentation for binary data handling
