# Sources

Reference material and documentation sources for this project. These were selected to support implementing dense printable encodings (base62, base85/Z85, base91), baseline comparisons (base64), and charset selection rules for custom high-density encodings.

- https://developer.mozilla.org/en-US/docs/Glossary/Base64
- https://datatracker.ietf.org/doc/html/rfc4648
- https://en.wikipedia.org/wiki/Base64
- https://en.wikipedia.org/wiki/Ascii85
- https://rfc.zeromq.org/spec:32/Z85/
- https://base91.sourceforge.net/
- https://en.wikipedia.org/wiki/Base62
- https://www.npmjs.com/package/base62
- https://www.npmjs.com/package/base91
- https://en.wikipedia.org/wiki/ASCII#Printable_characters
- https://datatracker.ietf.org/doc/html/rfc4122
- https://datatracker.ietf.org/doc/html/draft-peabody-dispatch-new-uuid-format
- https://www.npmjs.com/package/ascii85
- https://www.npmjs.com/package/z85
- https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Uint8Array

These sources were chosen to provide authoritative specifications and implementation notes for: base64/base64url (RFC 4648 + MDN), printable ASCII character ranges and rules, Z85 (ZeroMQ spec), Ascii85, Base91, Base62 references, npm packages implementing those encodings, and the UUID v7 format draft used by the mission.
